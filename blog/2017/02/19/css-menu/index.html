CSS 下拉菜单

Feb 19th, 2017 7:40 pm

百度一下，你就知道

用 ul,li 标签做个常用的下拉菜单。

  <ul class="menu_ul">
    <li class="menu_li">
    <a class="menu_title" href="#">软件开发</a>
    <div class="menu_container">
        <div id="menu_line"></div>
        <a class="menu_cell" href="#">前端开发</a>
        <a class="menu_cell" href="#">iOS开发</a>
        <a class="menu_cell" href="#">Android开发</a>
    </div>
    </li>
  </ul>

一个小效果要写这么多CSS…

 .menu_ul{
    margin: 0px;
    padding: 0px;
    list-style-type: none;
    display: inline-block;
 }
  .menu_ul a{
    text-decoration: none;
    color: black;
  }

 .menu_title{
    padding: 5px;
    display: inline-block;
 }
 .menu_container{
    background-color: white;
    display: none;
    position: absolute;
    border-width: 1px;
    border-top-width: 0px;
    border-color: red;
    border-style: solid;
 }
.menu_cell{
    display: block;
    padding: 10px;
}
 .menu_ul:hover .menu_title{
    background-color: white;
    border-width: 1px;
    border-bottom-width: 0px;
    border-color: red;
    border-style: solid;
 }

.menu_ul:hover .menu_container{
    display: block;
}

.menu_ul:hover a:link{
    color: black;
 }
 .menu_ul:hover a:visited{
     color: black;
 }
 .menu_ul:hover a:hover{
     color: red;
     text-decoration: underline;
     text-decoration-color: red;
 }
 .menu_ul:hover a:active{
       color: black;
 }

/*这条线暂时不知有啥好办法处理*/
 #menu_line{
    height: 1px;
    width: 40px;
    float: right;
    background-color: red;
 }
